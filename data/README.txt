=====================================
Helper files and generated data files
=====================================

Different tables which were generated by cablemap and helper files
to understand or interpret cables.


cabledrum_uris.csv
------------------
Table of ``reference_id: cabledrum.net URI`` entries.
Generated by cablemap.


classified_by.csv
-----------------
Table of ``reference_id: person-who-classified-the-cable`` entries.
Generated by cablemap.


nondisclosure_deadline.csv
--------------------------
Table of ``reference_id: non-disclosure deadline`` entries.
Generated by cablemap.


plusd_cids.csv
--------------
Table of ``reference_id: PlusD canonical ID`` entries.
Generated by cablemap.


plusd_uris.csv
-----------------
Table of ``reference_id: PlusD URI`` entries.
Generated by cablemap.


refidc14n.csv
-------------
Table of ``reference_id: canonical id`` entries.
Generated by cablemap.


signers.csv
------------
Table of ``reference_id: signer`` entries.
Generated by cablemap.


subjects.csv
------------
Table of ``reference_id: subject`` entries.
Generated by cablemap.


transmissionids.csv
-------------------
Table of ``reference_id: transmission id`` entries.
Generated by cablemap.
